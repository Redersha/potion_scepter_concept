# potion_scepter_mod_concept
**药水权杖（Potion Scepter）模组 - 完整设定**
关于药水权杖系统的构想，Minecraft JE模组

**提示：所有数值是瞎编来的，仅供参考**

## 核心机制总结
- **混合药水**：药锅混合，效果叠加，相反效果抵消
- **药水权杖**：双效果槽存储，左键发射/右键散布，自动影响玩家与敌对生物，中立生物需标记
- **权杖台**：装填效果槽，6条独立升级路线（每杖选1条，不可回头）

## 经验与耐久系统

### 经验消耗算法（右键散布时）
```
基础消耗 = 1级
效果消耗 = 0.1 × 效果等级
时间消耗 = 0.01 × 散布持续时间（秒）
总消耗 = 基础消耗 + 效果消耗 + 时间消耗（向上取整）

示例：散布速度II（3秒）：
1 + 0.1×2 + 0.01×3 = 1 + 0.2 + 0.03 = 1.23 → 消耗2级经验
```
- **保护机制**：玩家经验≤10级时，消耗量减半
- **左键发射不消耗经验**

### 击杀回复 - 临时经验系统
```
回复量 = 基础值 + 效果贡献度
基础值 = 2级
效果贡献度 = 0.5 × 负面效果等级

持续时间 = 30 + 5 × 回复量（秒）
```
- **生效条件**：权杖任一负面效果导致击杀/助攻
- **独立系统**：临时经验不与原版经验叠加，用于权杖升级/施法
- **可叠加**：多次击杀可延长持续时间，但上限300秒

### 耐久度系统
```
初始耐久 = 100
施法消耗：
- 左键发射：消耗1耐久
- 右键散布：消耗2耐久
- 标记生物：消耗0.5耐久

损坏阈值 = 20耐久
低于阈值时：
1. 施法有(20-当前耐久)%几率失败
2. 继承成功率下降
```

### 继承系统
```
继承成功率 = 100 - (当前耐久 / 初始耐久 × 100)%
成功率加成 = 10 × 继承次数（每杖可继承一次）

示例：
耐久30/100的权杖，首次继承：
100 - (30/100×100) = 70% 成功率
```
- **继承规则**：
  - 在权杖台内，新旧权杖+对应升级材料
  - 可选择继承已走“岔路”的任一阶段
  - 继承后旧权杖消失，新权杖获得旧权杖的升级路线与等级
  - 可额外消耗经验提高成功率（1级经验=+5%成功率）

## 喷溅药水机制
- **量化单位**：1秒效果时间 = 1“剂量”
- **基础喷射量**：每秒喷射10剂量
- **可升级**：在权杖台增加每秒喷射量
- **作用方式**：
  - 消耗存储剂量，瞬间对范围内生物施加完整效果
  - 3秒后效果消失（与剂量无关）
  - 范围固定为半径5格

## 平衡性注意事项
1. **经验消耗曲线**：前期友好，后期显著
2. **耐久限制**：鼓励玩家规划继承时机
3. **路线唯一性**：增加重复可玩性
4. **临时经验**：鼓励使用负面效果，平衡消耗
5. **喷射量限制**：避免喷溅药水过于强大

## 🎮 游戏内体验流程
1. **早期**：制作基础权杖，尝试混合药水
2. **中期**：选择升级路线，规划继承策略
3. **后期**：最大化路线收益，利用临时经验系统
4. **终局**：尝试不同路线组合，优化施法策略
